# Kalkulator Segitiga

Program Python sederhana untuk menghitung **luas** atau **keliling** dari segitiga berdasarkan pilihan dan input pengguna.

## Fitur

- Memilih mode perhitungan: Keliling (K) atau Luas (L)
- Input sisi-sisi segitiga untuk menghitung keliling
- Input panjang alas dan tinggi untuk menghitung luas
- Output hasil perhitungan langsung ditampilkan

## Cara Menjalankan

Pastikan Python 3 sudah terinstal di perangkat Anda.

Jalankan program dengan perintah berikut di terminal atau command prompt:

```bash
python Kalkulator\ Segitiga.py
```

Ikuti petunjuk:
- Ketik `K` untuk menghitung keliling
- Ketik `L` untuk menghitung luas
- Masukkan nilai-nilai yang diminta

## Contoh Output

```
=========================================
          Kalkulator Segitiga
=========================================
Keliling (K) atau Luas (L): K
Kamu memilih Keliling
Masukan Sisi Satu: 3
Masukan Sisi Dua: 4
Masukan Sisi Tiga: 5
Jadi Keliling Nya Adalah 12.0
```

Atau:

```
Keliling (K) atau Luas (L): L
Kamu memilih Luas
Masukan Panjang Alas: 6
Masukan Tinggi: 4
Jadi Luas Nya Adalah 12.0
```
